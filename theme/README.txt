Linen — Rockbox Theme for iPod Video (5G)
==========================================

A clean, original Rockbox theme designed at the iPod's native 320×240.
Warm-light palette, rounded geometric type (Nunito), text-forward Now
Playing screen.

Target
------
- iPod Video (5G) — 320×240, 16-bit color

Installation
------------
Copy the .rockbox folder to the root of your iPod (merge with existing).
Then on the device:
  Settings → Theme Settings → Browse Theme Files → linen

Fonts
-----
Includes Nunito at 9, 11, 13 (medium + bold), 17 (bold) px.
